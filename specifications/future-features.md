# Future Features

**Document Version:** 1.0  
**Status:** Planned  
**Last Updated:** 14/07/2026

---

# Purpose

This document lists features intentionally excluded from the MVP but considered for future releases.

The existence of a feature in this document does **not** imply that it will be implemented. It serves as a roadmap for possible future evolution while keeping the MVP focused.

---

# Guiding Principle

Future features should enhance the platform without compromising the project's core principles:

- Simplicity
- Maintainability
- Performance
- Business workflow preservation
- Mobile-first experience
